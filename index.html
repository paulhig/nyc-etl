About the data

The NYC 311 Service Requests dataset contains every non-emergency service request made to New York City from 2010 to present — over 35 million records. Each row represents a single complaint event with a timestamp, complaint type, responding agency, borough, zip code, and lat/lng coordinates. Data is served via the Socrata Open Data API (no API key required for up to 1,000 rows per request).
